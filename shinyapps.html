Shiny Applications 

MAVIS

MAVIS (Meta-Analysis via Shiny) is a web application and R package for running a meta-analysis with Shiny and R.

Available on CRAN
Code on GitHub
Online Demo http://kylehamilton.net/shiny/MAVIS/

 aRma

aRma is the Turkish version of MAVIS. Burak Aydin at Recep Tayyip Erdoğan University in Rize Turkey translated and maintains this application and part of the MAVIS R package.

Online Demo http://kylehamilton.net/shiny/aRma/


lavaan.shiny

lavaan.shiny (Latent Variable Analysis with Shiny) is a Shiny wrapper to the lavaan package. Currently lavaan.shiny allows users to run confirmatory factor analysis, growth curve models, and structural equation models.

Available on CRAN
Code on GitHub
Online Demo http://kylehamilton.net/shiny/lavaan.shiny/

CTTShiny

CTTShiny is a Shiny application for running classical test theory and item analysis with Shiny.

Available on CRAN
Code on GitHub
Online Demo http://kylehamilton.net/shiny/CTTShiny/


IRTShiny

IRTShiny is a Shiny application for running item

PACA
